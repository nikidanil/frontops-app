# 🚀 Приложение на Node.js

> 💡 **Цель проекта:** отработка навыков работы с Docker, Docker Compose, публикацией образов на Docker Hub и деплоем приложения на VPS-сервере.

## 🎯 Цель репозитория

Этот репозиторий предназначен для практического освоения процесса контейнеризации Node.js-приложения с помощью Docker и управления им через Docker Compose. Также проект демонстрирует:

- Сборку Docker-образа
- Публикацию образа на [Docker Hub](https://hub.docker.com/r/nikidanil/node-app)
- Подготовку к автоматизированному деплою на VPS-сервер

## 📦 Функционал приложения

Приложение представляет собой простой HTTP-сервер на Express, который:

- Отвечает на запросы к корневому маршруту `/`
- Возвращает сообщение: `Приложение работает!` с текущей датой и временем в формате: `вторник, 17 апреля 2026 г., 00:00:00 UTC` (на русском языке)
- Запускается на порту `3000`

## 📂 Структура проекта

- `index.js` — основной файл приложения (Express-сервер)
- `package.json` — зависимости и метаданные
- `Dockerfile` — инструкции для сборки образа
- `docker-compose.yml` — конфигурация сервиса
